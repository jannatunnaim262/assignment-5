Question-1: What are the different ways to select an element in the DOM?

We can select element by using ID,className,tagName,QuerySelectors etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

In innerHTML we can work with html content. In innerText we can show visible text. In textContent we can use all text and hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where parent element handles events for child elements.

Question-4: What is event bubbling in the DOM?

Its a technique where an events starts at the target element and listen clicks anywhere in the page.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create html element using document.createElement(), add element using appendChild(), remove element using remove().
